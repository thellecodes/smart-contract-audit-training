# Threat Model

This provides a full threat model description for various functions. It analyzes each function within the smart contracts and presents a written threat model for critical functions. A threat model documents a given function's externally controllable inputs and how an attacker could leverage each input to cause harm.

Not all functions may have been modeled. The absence of a threat model in this section does not necessarily suggest that a function is safe.
